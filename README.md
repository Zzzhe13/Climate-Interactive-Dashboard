# Climate-Interactive-Dashboard
SDU VIS Final Project

我们选择中国近十年的气候数据情况作为我们的数据集，在温度，湿度，风速，降水量，空气质量，实时天气，这六个方面来进行可视化呈现，用户可以交互的筛选数据，旨在呈现多样气候趋势，分布等等情况的互动可视化
